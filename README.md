# 香港共和國 · 官方網站（Hong-Kong Republic Official Website）

> **Repository Type:** Official Website (Static Site)  
> **Deployment Target:** GitHub Pages  
> **Tech Stack (planned):** Astro + Markdown Content Collections

---

## 1. 目的（Purpose）
本倉庫用於構建「香港共和國（Hong-Kong Republic）」的官方網站，提供以下內容：

- 官方公告（Press Releases）
- 憲政架構與政府部門介紹
- 國史年表（Timeline）
- 重要文件（如《維多利亞協議》設定文本、獨立宣言等）
- 旅客與外國使團資訊（可選）
- 國家象徵（國旗 / 國徽 / 格言）

此專案為**虛構世界觀設定**用途，內容不代表現實政治立場。

---

## 2. 部署（Deployment）
本網站將使用 GitHub Pages 進行部署。

部署方式（後續會配置）：
- 使用 GitHub Actions 自動構建
- 將生成的靜態頁面發佈到 Pages

> 目前尚未初始化前端框架，後續會一步步添加。

預計流程：
1. 初始化 Astro 專案
2. 建立網站基礎架構（Layout / 路由）
3. 加入內容系統（Markdown Collections）
4. 設計官方視覺（色彩 / 字體 / 國旗國徽 SVG）

---

## 4. 內容結構（Planned Information Architecture）
網站將包含（草案）：

- `/` 首頁：國家簡介、最新公告、快速入口
- `/government` 政府：制度、部門、官職
- `/laws` 法律：憲法摘要、核心法規（世界觀文本）
- `/timeline` 年表：重大事件與年代記
- `/press` 公告：新聞稿、政策發布、外交聲明
- `/symbols` 國家象徵：旗、徽、格言、國歌（如需要）
